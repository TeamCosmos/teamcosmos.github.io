# 프로그래밍 학습 사이트 소개

##### 2016 제주대학교 오픈소스개발방법론 팀 페이지입니다.

#### 기회조차 얻지 못해서 그 능력을 무시당하는 일은 있어서는 안된다는 선조들의 말이 있었습니다.
지금 인터넷에는 다양한 언어를 학습하고 직접 코딩해 볼 수 있는 사이트가 많이 있지만 대부분이 영어로 되어 있거나 그 존재를 사람들이 알지못하여 이용하지 못하는 경우가 빈번히 발생하고 있습니다.
저희는 그러한 사이트를 소개하고 간단한 이용 방법을 알려줌으로써 보다 좋은 프로그램이 보다 쉽게 세상에 나올 수 있도록 하는 발판을 마련해 봤습니다.
